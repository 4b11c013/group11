# 音樂檔案放置說明

請將您的 MP3 音樂檔案放在對應的資料夾中：

## Lo-Fi 音樂包
- `lofi/beat.mp3` - 節拍
- `lofi/piano.mp3` - 鋼琴
- `lofi/bass.mp3` - 貝斯

## Cyberpunk 音樂包
- `cyberpunk/synth.mp3` - 合成器
- `cyberpunk/drums.mp3` - 鼓組
- `cyberpunk/bass.mp3` - 貝斯

## House 音樂包
- `house/kick.mp3` - 底鼓
- `house/hihat.mp3` - 高音鈸
- `house/bassline.mp3` - 貝斯線

將檔案放在對應的資料夾後，在應用程式中選擇對應的音樂包即可載入。
